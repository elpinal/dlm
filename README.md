# Dlm

`dlm` is a download manager.

## Install

To install, use `go get`:

```bash
$ go get -u github.com/elpinal/dlm
```

-u flag stands for "update".

## Examples

```bash
$ dlm example.com golang.org/pkg/path/filepath golang.org/ref/spec
1270
100% 28339/28339
100% 219533/219533
```
